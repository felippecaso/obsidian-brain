# Keyboard Layout
## Fn Layer
### Accented characters
- ç - `Fn1 + C` (M0)
- á - `Fn1 + A` (M1)
- Á - `Fn3 + A` (M11)
- é - `Fn1 + E` (M2)
- É - `Fn3 + E` (M12)
- í - `Fn1 + I` (M3)
- Í - `Fn3 + I` (M13)
- ó - `Fn1 + O` (M4)
- Ó - `Fn3 + O` (M14)
- ú - `Fn1 + U` (M5)
- Ú - `Fn3 + U` (M15)
- ã - `Fn + N` (M6)
- õ - `Fn + M` (M7)
- à - `Fn + Q` (M8)
- â - `Fn + Z` (M9)
- ê - `Fn + W` (M10)
- ô
